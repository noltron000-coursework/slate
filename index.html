A list of all of the HTML tags.

  • Comment Tag

    Describes code. Hidden from user.

    <!-- ...This is a comment... -->

  • !DOCTYPE Tag

    Defines document type. Hidden from user.

    <!DOCTYPE HTML />

  • Anchor Tags

    This tag creates a hyperlink, which redirects to another page on click.

    <a href='example.com'>example.com</a>
    example.com

  • Abbreviation Tag

    Defines an acronym or abbreviation.

    <abbr>TL;DR</abbr>
    TL;DR

  • Address Tag

    Defines contact information for author/owner of a document.

    <address>555 Post Street</address>
    555 Post Street

  • Area Tag

    Defines an area inside an image-map.

    <area>TL;DR</area>

  • Article Tag

    *NEW* Defines an article.

    <article>Rest In Peace</article>
    Rest In Peace

  • Aside Tag

    *NEW* Defines content to the side of the page content.

    <aside>I'm on the side.</aside>
    I'm on the side.

  • Audio Tag

    *NEW* Defines sound content.

    <audio>I'm Music!</audio>

  • Bold Tag

    Defines bold text.

    <b>I'm bold and daring.</b>
    I'm bold and daring.

  • Base Tag

    specificies the base url/target for all relative urls in a document.

    <b>I'm bold and daring.</b>
    I'm bold and daring.
